staticKakeraDOM.debounce(func, wait, immediate)

설정한 function을 일정한 시간 내에 한 번만 실행되도록한다.
Name Type Description
func function 설정하고 싶은 function.
wait Number 주기(ms).
immediate Boolean 즉각 실행여부.
Example
// 스크롤 이벤트에 대한 callback처리를 30ms마다 한다.
document.getElementById("myDIV").addEventListener(
  "scroll",
  KakeraDOM.debounce(function () { console.log("scroll...") }, 30, false)
);

보간(Interpolation)을 활용한 문자열 format.
Name Type Description
format String 보간이 삽입되어 있는 문자열.
args Array | Object | String 치환될 값.
Returns:
Type Description
String 변환된 문자열.
Example

// 철이와 미애

KakeraDOM.format("제 이름은 {name}이고 나이는 {age}입니다.", {name: "철이", age: "21"});
// 제 이름은 철이이고 나이는 21입니다.

KakeraDOM.format("제 이름은 {0}이고 나이는 {0}입니다.", ["철이", 21]);
// 제 이름은 철이이고 나이는 21입니다.
