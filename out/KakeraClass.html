KakeraClass

new KakeraClass()

KakeraClass.
Kakera framework의 component를 정의하는 class.

Members

depsArray

종속성을 설정한다. 설정시 array로 할 것.
Default Value:
  • null
Example
...
<div id='target'>
...
<script>
// Main 영역에 Sub를 2개 추가.
// Main은 Sub가 없다면 정상적으로 동작하지 않으므로 deps에 Sub를 설정해줘야 한다.
var Sub = KakeraDOM.createClass("Sub", {
  template: "<div>sub component입니다.</div>"
});
var Main = KakeraDOM.createClass("Main", {
  deps: [Sub],
  template: "<div>여기는 Main입니다.\
    <Sub />\
    <Sub />\
  </div>\"
});
KakeraDOM.render(
  new Main(),
  document.getElementById("target")
);
</script>

elElement

template또는 render에 의해 정의된 element. lifecycle에서 init을 제외한 나머지 function에서 접근 가능하다.
See:
  • KakeraClass#template
  • KakeraClass#render
Example
...
<div id='target'>
...
<script>
var HelloKakera = KakeraDOM.createClass("HelloKakera", {
  template: "<p></p>",
  willMount: function () {
    this.el.innerText = "hello kakera";
  }
});
KakeraDOM.render(
  new HelloKakera(),
  document.getElementById("target")
);
</script>

mixinsArray

여러 component에 동일한 behavior를 갖도록 한다.

// 다음은 늑대인간과 강아지 로봇을 동일한 행위를 하도록 설계/구현한 예제이다.

    this.swingTail();
  }

var PuppyRobot = KakeraDOM.createClass("PuppyRobot", {
  mixins: [DogBehavior],

  }
});

addChild(child)

child component를 추가한다.
Name Type Description
child KakeraClass child로 추가할 component.
Example
...
<div id='target'></div>
...
<script>
var Main = KakeraDOM.createClass("Main", {
 template: "<div>여기는 main</div>",
 didMount: function () {
   var Sub = KakeraDOM.createClass("Sub", {
     template: "<div>여기는 sub{this.props.idx}</div>"
   });
   for (var i = 0; i < 10; i++) {
     var sub = new Sub({idx: i});
     this.addChild(sub);	// child component 추가.
     KakeraDOM.render(sub, this.el);
   }
 }
});
KakeraDOM.render(
  new Main(),
  document.getElementById("target")
);
</script>

destroy()

(!주의 - 직접 호출하지 말 것 ) release function에 의해 호출된다. component가 더이상 사용되지 않으므로 listener 또는 timer 등을 이 function에서 제거한다.
Example
...
<div id='target'></div>
...
<script>
var Clock = KakeraDOM.createClass("Clock", {
  template: "<div>Clock component</div>",
  init: function () {
    this._interID = setInterval(function () { console.log(new Date()); }, 1000);
  },
  didMount: function () {
    setTimeout(function () { this.release(); }.bind(this), 3000);	// 3초 후 이 컴포넌트를 release한다.
  },
  destroy: function () {
    clearInterval(this._interID);
  }
});
KakeraDOM.render(
  new Clock(),
  document.getElementById("target")
);
</script>

render(){String|Ojbect}

template만으로 표현하기 힘든 HTML을 처리한다.
Returns:
Name Type Description
String | Ojbect
ret String HTML 문자열.
ret Object HTML 정보.
Name Type Description
template String HTML 문자열(보간 포함 가능).
param Object ret.template에 보간이 사용되었다면 치환되어야할 정보.
Example
...
<div id='customerList'></div>
...
<script>
var customers = [{name: "철이", age: 21}, {name: "미애", age:20}];

var List = KakeraDOM.createClass("List", {
  render: function () {
    var ret = "";
    var format = "<li>이름:{name} 나이:{age}</li>";
    for (var i = 0; i < customers.length; i++) {
      ret += KakeraDOM.format(format, customers[i]);
    }
    ret = "<ul>" + ret + "</ul>";
    return ret;
  }
});
KakeraDOM.render(
  new List(),
  document.getElementById("customerList")
);

var Over21Age = KakeraDOM.createClass("Over21Age", {
  render: function () {
    var param;
    for (var i = 0; i < customers.length; i++) {
      if (customers[i].age >= 21) {
        param = customers[i];
        break;
      }
    }
    return {
      template: "<div>21이상은 사람은 {name}입니다.</div>",
      param: param
    };
  }
});
KakeraDOM.render(
  new Over21Age(),
  document.getElementById("customerList")
);
</script>
